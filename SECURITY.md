# Security Policy

## Supported Versions

Finstack Quant is currently pre-1.0. Security fixes are provided for the latest
released version only.

| Version | Supported |
|---|---|
| latest | yes |
| older versions | no |

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by emailing
me@jeickmeier.com. Do not open a public issue for suspected vulnerabilities.

Include:

- Affected package, crate, or module.
- A minimal reproduction or proof of concept when possible.
- Impact assessment and any known workaround.
- Whether the issue is already public.

Reports will be reviewed privately before disclosure or public remediation.
